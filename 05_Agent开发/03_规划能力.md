## ReAct框架

### 什么是ReAct？
**核心思想**：推理（Reasoning）+ 行动（Acting）

### 工作流程
```
问题：北京今天天气怎么样？

思考：我需要查询北京的天气信息
行动：调用天气API，参数：城市=北京
观察：北京今天晴天，25度
思考：我已经获得了天气信息
答案：北京今天晴天，温度25度
```

## 思维链规划

### 什么是思维链规划？
**定义**：使用思维链分解复杂任务

### 规划提示
```python
planning_prompt = """
请为以下任务制定详细计划：

任务：{task}

要求：
1. 分解为子任务
2. 确定子任务顺序
3. 估计每个子任务所需时间
4. 确定所需资源

输出格式：
子任务1：{描述}，预计时间：{时间}
子任务2：{描述}，预计时间：{时间}
...
"""
```

### 代码实现
```python
def plan_task(task, llm):
    """使用思维链规划任务"""
    
    prompt = planning_prompt.format(task=task)
    plan = llm.generate(prompt)
    
    # 解析计划
    subtasks = parse_plan(plan)
    
    return subtasks

def parse_plan(plan_text):
    """解析计划文本"""
    lines = plan_text.strip().split('\n')
    subtasks = []
    
    for line in lines:
        if line.startswith('子任务'):
            parts = line.split('：')
            if len(parts) >= 2:
                subtasks.append({
                    'description': parts[1].split('，')[0],
                    'time': parts[1].split('，')[1] if '，' in parts[1] else '未知'
                })
    
    return subtasks
```

## 任务分解

### 分解策略
```
1. 按步骤分解
   复杂任务 → 简单步骤

2. 按组件分解
   系统任务 → 组件任务

3. 按目标分解
   最终目标 → 子目标
```

### 代码实现
```python
def decompose_task(task, decomposition_type="steps"):
    """分解任务"""
    
    if decomposition_type == "steps":
        return decompose_by_steps(task)
    elif decomposition_type == "components":
        return decompose_by_components(task)
    elif decomposition_type == "goals":
        return decompose_by_goals(task)

def decompose_by_steps(task):
    """按步骤分解"""
    prompt = f"""
    将以下任务分解为具体的执行步骤：
    
    任务：{task}
    
    请列出所有必要的步骤，每个步骤一行：
    """
    
    steps = llm.generate(prompt).strip().split('\n')
    return [step.strip() for step in steps if step.strip()]
```

## 规划评估

### 评估指标
```python
def evaluate_plan(plan, task):
    """评估计划质量"""
    
    criteria = {
        "completeness": 0,  # 完整性
        "feasibility": 0,   # 可行性
        "efficiency": 0,    # 效率
        "clarity": 0        # 清晰度
    }
    
    # 检查是否覆盖所有必要步骤
    necessary_steps = get_necessary_steps(task)
    covered_steps = [s for s in necessary_steps if s in plan]
    criteria["completeness"] = len(covered_steps) / len(necessary_steps)
    
    # 检查步骤是否可行
    feasible_steps = [s for s in plan if is_feasible(s)]
    criteria["feasibility"] = len(feasible_steps) / len(plan)
    
    # 检查步骤顺序是否合理
    criteria["efficiency"] = evaluate_efficiency(plan)
    
    # 检查步骤描述是否清晰
    criteria["clarity"] = evaluate_clarity(plan)
    
    return criteria
```

## 自适应规划

### 根据反馈调整
```python
class AdaptivePlanner:
    def __init__(self, llm):
        self.llm = llm
        self.original_plan = None
        self.current_plan = None
        self.feedback_history = []
    
    def create_initial_plan(self, task):
        """创建初始计划"""
        self.original_plan = self.generate_plan(task)
        self.current_plan = self.original_plan.copy()
        return self.current_plan
    
    def adjust_plan(self, feedback, error_info=None):
        """根据反馈调整计划"""
        
        prompt = f"""
        原始计划：{self.original_plan}
        
        当前计划：{self.current_plan}
        
        反馈：{feedback}
        
        错误信息：{error_info or '无'}
        
        请调整计划以解决问题：
        """
        
        adjusted_plan = self.llm.generate(prompt)
        self.current_plan = parse_plan(adjusted_plan)
        self.feedback_history.append(feedback)
        
        return self.current_plan
```
