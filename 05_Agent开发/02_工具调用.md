# 工具调用

## 什么是工具调用？

### 一句话定义
**一句话**：让LLM能够使用外部工具（API、函数等）

### 类比理解
```
人：遇到数学题，用计算器算
Agent：遇到数学题，调用计算工具

人：需要天气信息，查看天气APP
Agent：需要天气信息，调用天气API
```

## 工具调用原理

### 工作流程
```
1. LLM分析任务需求
2. LLM决定调用哪个工具
3. LLM生成工具参数
4. 执行工具函数
5. 返回结果给LLM
6. LLM基于结果继续推理
```

### 工具调用示例
```
用户问题：北京今天天气怎么样？

LLM思考：我需要查询北京的天气信息
LLM决定：调用get_weather工具
LLM参数：city="北京"
工具执行：get_weather("北京")
工具返回："北京今天晴天，25度"
LLM回答：北京今天晴天，温度25度
```

## 工具定义

### 基础工具定义
```python
from langchain.tools import tool

@tool
def calculator(expression: str) -> str:
    """计算数学表达式"""
    return str(eval(expression))

@tool
def search_web(query: str) -> str:
    """搜索互联网获取信息"""
    return f"搜索结果：关于{query}的信息..."

@tool
def get_weather(city: str) -> str:
    """获取城市天气"""
    return f"{city}今天晴天，25度"
```

### 高级工具定义
```python
from langchain.tools import Tool
from pydantic import BaseModel, Field

class WeatherInput(BaseModel):
    city: str = Field(description="城市名称")
    date: str = Field(description="日期，格式：YYYY-MM-DD", default="today")

def get_weather(city: str, date: str = "today") -> str:
    """获取指定城市和日期的天气"""
    # 模拟API调用
    return f"{city} {date} 天气：晴天，温度25度"

# 创建工具
weather_tool = Tool(
    name="get_weather",
    func=get_weather,
    description="获取指定城市和日期的天气信息",
    args_schema=WeatherInput
)
```

## 工具集成

### 创建Agent
```python
from langchain.agents import AgentExecutor, create_openai_functions_agent
from langchain.chat_models import ChatOpenAI
from langchain.prompts import ChatPromptTemplate

# 1. 创建LLM
llm = ChatOpenAI(model="gpt-3.5-turbo", temperature=0)

# 2. 定义工具
tools = [calculator, search_web, get_weather]

# 3. 创建提示模板
prompt = ChatPromptTemplate.from_messages([
    ("system", "你是一个有用的助手，可以使用工具来回答问题。"),
    ("human", "{input}"),
    ("agent_scratchpad", "{agent_scratchpad}")
])

# 4. 创建Agent
agent = create_openai_functions_agent(llm, tools, prompt)
agent_executor = AgentExecutor(agent=agent, tools=tools, verbose=True)

# 5. 运行Agent
result = agent_executor.invoke({"input": "北京天气怎么样？"})
print(result["output"])
```

### 2. API工具
```python
import requests

@tool
def get_stock_price(symbol: str) -> str:
    """获取股票价格"""
    url = f"https://api.example.com/stock/{symbol}"
    response = requests.get(url)
    return response.json()["price"]
```

### 3. 数据库工具
```python
import sqlite3

@tool
def query_database(sql: str) -> str:
    """执行SQL查询"""
    conn = sqlite3.connect("database.db")
    cursor = conn.cursor()
    cursor.execute(sql)
    results = cursor.fetchall()
    conn.close()
    return str(results)
```

### 4. 文件工具
```python
@tool
def read_file(file_path: str) -> str:
    """读取文件内容"""
    with open(file_path, 'r') as f:
        return f.read()

@tool
def write_file(file_path: str, content: str) -> str:
    """写入文件"""
    with open(file_path, 'w') as f:
        f.write(content)
    return "文件写入成功"
```

### 5. 代码执行工具
```python
@tool
def execute_python(code: str) -> str:
    """执行Python代码"""
    try:
        # 安全执行（生产环境需要沙箱）
        exec_globals = {}
        exec(code, exec_globals)
        return str(exec_globals.get('result', '执行成功'))
    except Exception as e:
        return f"执行错误: {str(e)}"
```

## 工具优化

### 1. 错误处理
```python
@tool
def safe_api_call(query: str) -> str:
    """安全的API调用"""
    try:
        result = api_call(query)
        return result
    except APIError as e:
        return f"API错误: {str(e)}"
    except TimeoutError:
        return "请求超时，请稍后重试"
    except Exception as e:
        return f"未知错误: {str(e)}"
```

### 2. 缓存机制
```python
from functools import lru_cache
import time

@lru_cache(maxsize=100)
def cached_api_call(query: str) -> str:
    """带缓存的API调用"""
    return api_call(query)

# 带TTL的缓存
class TTLCache:
    def __init__(self, ttl_seconds=300):
        self.cache = {}
        self.ttl = ttl_seconds
    
    def get(self, key):
        if key in self.cache:
            value, timestamp = self.cache[key]
            if time.time() - timestamp < self.ttl:
                return value
        return None
    
    def set(self, key, value):
        self.cache[key] = (value, time.time())
```

## 工具链

### 创建工具链
```python
from langchain.chains import SequentialChain
from langchain.chains import TransformChain

# 定义处理步骤
def parse_input(inputs):
    """解析输入"""
    inputs["parsed"] = inputs["input"].split()
    return inputs

def process_words(inputs):
    """处理单词"""
    inputs["processed"] = [w.upper() for w in inputs["parsed"]]
    return inputs

def format_output(inputs):
    """格式化输出"""
    inputs["output"] = ", ".join(inputs["processed"])
    return inputs

# 创建链
chain = SequentialChain(
    chains=[
        TransformChain(
            input_variables=["input"],
            output_variables=["parsed"],
            transform=parse_input
        ),
        TransformChain(
            input_variables=["parsed"],
            output_variables=["processed"],
            transform=process_words
        ),
        TransformChain(
            input_variables=["processed"],
            output_variables=["output"],
            transform=format_output
        )
    ],
    input_variables=["input"],
    output_variables=["output"]
)

# 运行链
result = chain.run("hello world test")
print(result)  # HELLO, WORLD, TEST
```

## 最佳实践

### 1. 工具设计
```
原则：
  - 单一职责：每个工具只做一件事
  - 明确描述：提供清晰的工具描述
  - 参数验证：验证输入参数
  - 错误处理：优雅处理错误
```

### 2. 安全性
```
措施：
  - 输入验证：防止注入攻击
  - 权限控制：限制工具访问范围
  - 沙箱执行：代码执行使用沙箱
  - 日志记录：记录工具使用日志
```

### 3. 性能优化
```
方法：
  - 缓存结果：避免重复计算
  - 异步执行：长时间操作异步
  - 批量处理：合并相似请求
  - 连接池：复用数据库连接
```
