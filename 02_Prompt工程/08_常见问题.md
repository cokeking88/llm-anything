# 常见问题解答

## 基础问题

### Q1：Prompt越长越好吗？
**不一定**：
- 简单任务：短prompt即可
- 复杂任务：需要详细prompt
- 注意token限制和成本
- 平衡：足够详细但不冗余

### Q2：如何提高回答质量？
**技巧**：
1. 给AI"思考时间"（思维链）
2. 提供上下文
3. 指定角色
4. 要求自我验证
5. 使用少样本学习
6. 添加约束条件
7. 分步指导

### Q3：什么是思维链？为什么有效？
**思维链**：让模型展示推理过程
**有效原因**：
- 强制模型进行逐步推理
- 减少跳跃性错误
- 提高复杂问题解决能力
- 增强可解释性

## 技术问题

### Q4：如何控制输出格式？
**方法**：
1. 明确指定格式：JSON、XML、Markdown
2. 提供格式示例
3. 使用模板填充
4. 添加格式验证

### Q5：温度参数如何设置？
**建议**：
- 事实性任务：0.1-0.3（确定性）
- 创意任务：0.7-1.2（创造性）
- 通用任务：0.7（平衡）
- 调试：0（完全确定）

### Q6：如何处理模型幻觉？
**策略**：
1. 要求提供来源
2. 使用RAG技术
3. 添加事实核查步骤
4. 降低温度
5. 使用思维链
6. 后处理验证

## 实践问题

### Q7：如何设计好的角色？
**原则**：
1. 具体专业背景
2. 明确知识范围
3. 指定回答风格
4. 设定行为边界
5. 包含示例

### Q8：少样本学习给几个例子？
**建议**：
- 简单任务：1-2个
- 复杂任务：3-5个
- 格式控制：2-3个
- 多样性：覆盖不同情况

### Q9：如何优化失败的Prompt？
**步骤**：
1. 分析失败原因
2. 诊断问题类型
3. 针对性调整
4. 测试改进版本
5. 迭代优化

## 高级问题

### Q10：如何设计复杂任务Prompt？
**策略**：
1. 分解任务
2. 设计指令链
3. 添加验证步骤
4. 使用模板
5. 渐进式引导

### Q11：如何处理模糊需求？
**方法**：
1. 主动澄清
2. 提供选项
3. 分步确认
4. 使用对话式Prompt

### Q12：如何平衡创意和准确性？
**平衡方法**：
- 低温：高准确性，低创意
- 高温：高创意，低准确性
- 分层：先准确后创意
- 后处理：创意后验证

## 工具问题

### Q13：不同模型需要不同的Prompt吗？
**是的**：
- 不同模型有不同特点
- 需要根据模型调整
- 测试不同版本
- 记录模型特定优化

### Q14：如何测试Prompt效果？
**方法**：
1. 准备测试数据集
2. 定义评估指标
3. 运行测试
4. 分析结果
5. A/B测试

### Q15：如何管理Prompt库？
**建议**：
1. 版本控制
2. 分类组织
3. 文档说明
4. 使用统计
5. 定期维护
