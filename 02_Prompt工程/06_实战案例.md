# 实战案例

## 文本生成

### 案例1：文章写作
```
Prompt：
"你是一个技术博客作家。请写一篇关于'Transformer架构'的文章。

要求：
1. 目标读者：有基础编程知识的初学者
2. 长度：约1000字
3. 结构：引言、核心概念、工作原理、应用、总结
4. 风格：通俗易懂，使用类比
5. 包括：1-2个代码示例

输出格式：Markdown格式，有标题和小标题"
```

### 案例2：故事创作
```
Prompt：
"创作一个科幻短故事。

设定：
- 背景：2050年，AI已经普及
- 主角：一个AI伦理学家
- 冲突：发现一个AI产生了自我意识
- 主题：探讨AI权利和人类责任

要求：
1. 长度：约800字
2. 风格：悬疑，带点哲学思考
3. 结构：起承转合
4. 包括：生动的对话和场景描写

输出：完整的叙事文本"
```

## 代码生成

### 案例1：函数实现
```
Prompt：
"请用Python实现一个函数，功能是：计算两个日期之间的天数差。

要求：
1. 函数名：days_between_dates
2. 参数：date1 (字符串，格式：YYYY-MM-DD), date2 (字符串，格式：YYYY-MM-DD)
3. 返回值：整数，天数差（date2 - date1）
4. 处理：考虑闰年
5. 包含：类型注解和docstring
6. 包含：使用示例

输出：完整的Python代码"
```

### 案例2：代码解释
```
Prompt：
"请解释以下Python代码的功能和工作原理：

```python
def fibonacci(n):
    if n <= 1:
        return n
    a, b = 0, 1
    for _ in range(2, n + 1):
        a, b = b, a + b
    return b
```

要求：
1. 用简单语言解释
2. 说明算法思路
3. 分析时间复杂度
4. 给出使用示例

输出：结构化的解释，包括代码块"
```

## 数据分析

### 案例1：销售数据分析
```
Prompt：
"你是一个数据分析师。请分析以下销售数据：

数据：
日期,产品,销售额,数量
2024-01-01,产品A,1000,10
2024-01-01,产品B,2000,5
2024-01-02,产品A,1200,12
2024-01-02,产品B,1800,4

要求：
1. 计算总销售额和总数量
2. 分析产品表现差异
3. 找出销售趋势
4. 给出业务建议

输出格式：
## 数据概览
## 详细分析
## 关键发现
## 业务建议"
```
