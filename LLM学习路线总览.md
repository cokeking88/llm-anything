# LLM大模型学习路线总览

## 什么是大语言模型（LLM）？

**一句话**：读了海量文本后学会"说话"的AI

**类比**：
```
传统AI：教小孩做数学题（规则明确）
大模型：教小孩读万卷书后写文章（从数据中学习）

ChatGPT = 读了互联网上几乎所有文字后学会的"超级学生"
```

## 核心概念速查

| 概念 | 一句话解释 |
|------|------------|
| Transformer | 基于注意力机制的架构 |
| Token | 文字被拆成的小块 |
| 预训练 | 在海量文本上学习语言规律 |
| 微调 | 在特定任务上进一步训练 |
| LoRA | 低秩适配，省资源微调 |
| RAG | 先检索文档，再生成答案 |
| Agent | 能思考、能行动的AI助手 |
| 多模态 | 能处理文字、图片、声音 |
