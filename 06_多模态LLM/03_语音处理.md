# 语音处理

## 语音识别

### 什么是语音识别？
**定义**：将语音转换为文字

### 主流模型
```
1. Whisper（OpenAI）
   - 支持多种语言
   - 开源可部署
   - 准确率高

2. wav2vec2（Meta）
   - 自监督学习
   - 多语言支持
   - 可微调
```

### 使用Whisper
```python
import whisper

# 加载模型
model = whisper.load_model("base")

# 转录音频
result = model.transcribe("audio.mp3")
print(result["text"])
```

## 语音合成

### 什么是语音合成？
**定义**：将文字转换为语音

### 主流模型
```
1. Tacotron2（Google）
   - 高质量合成
   - 自然度好

2. VITS
   - 端到端合成
   - 效率高
```

### 语音合成示例
```python
from transformers import VitsModel, AutoTokenizer
import torch

# 加载模型
model = VitsModel.from_pretrained("facebook/mms-tts-eng")
tokenizer = AutoTokenizer.from_pretrained("facebook/mms-tts-eng")

# 合成语音
text = "Hello, this is a test."
inputs = tokenizer(text, return_tensors="pt")

with torch.no_grad():
    outputs = model(**inputs)
    
# 保存音频
waveform = outputs.waveform
```

## 语音对话

### 语音对话系统
```
组件：
1. 语音识别：语音 → 文字
2. 文本处理：LLM处理文字
3. 语音合成：文字 → 语音

流程：
用户语音 → ASR → LLM → TTS → 语音回复
```
