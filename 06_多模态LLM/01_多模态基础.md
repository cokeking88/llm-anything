# 多模态基础

## 什么是多模态LLM？

### 一句话定义
**一句话**：能同时理解文字、图片、声音、视频的AI

### 类比理解
```
普通LLM：只能看文字
多模态LLM：像人一样，能看图、听声音、读文字

GPT-4o：能看图片、听语音、生成图片
```

## 主流多模态模型

### 模型对比
| 模型 | 公司 | 支持模态 | 特点 |
|------|------|----------|------|

```
  - 图像生成能力

Gemini 1.5：

  - 百万token上下文

LLaVA：
  - 开源模型

  - 可本地部署
```

## 多模态原理

### 架构设计
```
图片 → 图像编码器 → 图像Token
                              ↓
文字 → 文字编码器 → 文字Token → Transformer → 输出
音频 → 音频编码器 → 音频Token
```

### 图像理解流程
```
图片 → Vision Encoder → 图像特征向量
                              ↓
                         投影层
                              ↓
                        LLM处理
                              ↓
                         文字描述
```

### 多模态融合
```
1. 特征融合：不同模态特征合并
2. 注意力机制：跨模态注意力
3. 对齐学习：模态间对齐
4. 联合训练：多模态数据训练
```

## 使用多模态模型

### GPT-4V图片理解
```python
from openai import OpenAI

client = OpenAI()

response = client.chat.completions.create(
    model="gpt-4-vision-preview",
    messages=[
        {
            "role": "user",
            "content": [
                {"type": "text", "text": "这张图片里有什么？"},
                {
                    "type": "image_url",
                    "image_url": {
                        "url": "https://example.com/image.jpg"
                    }
                }
            ]
        }
    ]
)

print(response.choices[0].message.content)
```

### 使用开源模型LLaVA
```python
from transformers import LlavaNextProcessor, LlavaNextForConditionalGeneration
from PIL import Image

# 加载模型
processor = LlavaNextProcessor.from_pretrained("llava-hf/llava-v1.6-mistral-7b-hf")
model = LlavaNextForConditionalGeneration.from_pretrained("llava-hf/llava-v1.6-mistral-7b-hf")

# 加载图片
image = Image.open("image.jpg")

# 生成描述
prompt = "[INST] <image>\n描述这张图片 [/INST]"
inputs = processor(prompt, image, return_tensors="pt")
output = model.generate(**inputs, max_new_tokens=100)

print(processor.decode(output[0], skip_special_tokens=True))
```

## 多模态技术

### 视觉编码器
```
类型：Vision Transformer (ViT)
功能：将图片转换为特征向量
特点：捕捉图片中的视觉信息
```

### 跨模态对齐
```
目标：让不同模态的信息对齐
方法：对比学习、投影层
效果：图片和文字可以在同一空间表示
```

### 多模态生成
```
文字生成：基于图片生成文字描述
图片生成：基于文字生成图片（DALL-E、Stable Diffusion）
视频生成：基于文字生成视频
```

## 应用场景

### 视觉问答
```
场景：回答关于图片的问题
示例：
  输入：图片 + "这张图片里有什么？"
  输出："图片里有一只猫在沙发上睡觉"
```

### 图像描述
```
场景：为图片生成文字描述
示例：
  输入：图片
  输出："一只金色的狗在草地上奔跑，背景是蓝天白云"
```

### 视频理解
```
场景：理解视频内容
示例：
  输入：视频
  输出："视频展示了一个人在做饭的全过程"
```

### 多模态搜索
```
场景：基于图片搜索相似内容
示例：
  输入：图片
  输出：相似图片和相关信息
```

## 实战示例

### 图像描述系统
```python
def describe_image(image_path):
    # 加载图片
    image = Image.open(image_path)
    
    # 使用GPT-4V
    response = client.chat.completions.create(
        model="gpt-4-vision-preview",
        messages=[
            {
                "role": "user",
                "content": [
                    {"type": "text", "text": "详细描述这张图片"},
                    {"type": "image_url", "image_url": {"url": image_path}}
                ]
            }
        ]
    )
    
    return response.choices[0].message.content
```

### 视觉问答系统
```python
def visual_question_answering(image_path, question):
    # 使用多模态模型
    response = client.chat.completions.create(
        model="gpt-4-vision-preview",
        messages=[
            {
                "role": "user",
                "content": [
                    {"type": "text", "text": question},
                    {"type": "image_url", "image_url": {"url": image_path}}
                ]
            }
        ]
    )
    
    return response.choices[0].message.content
```
