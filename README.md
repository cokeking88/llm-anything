## 核心概念速查

| 概念 | 一句话解释 |
|------|------------|
| LLM | 读了海量文本后学会"说话"的AI |
| Transformer | 基于注意力机制的架构 |
| Token | 文字被拆成的小块 |
| 预训练 | 在海量文本上学习语言规律 |
| 微调 | 在特定任务上进一步训练 |
| LoRA | 低秩适配，省资源微调 |
| RAG | 先检索文档，再生成答案 |
| Agent | 能思考、能行动的AI助手 |
