# 训练流程

## 数据准备

### 数据格式
```json
{
  "instruction": "将以下英文翻译成中文",
  "input": "Hello, how are you?",
  "output": "你好，你好吗？"
}
```

### 数据类型
```
1. 指令数据
   格式：instruction + input + output
   示例：翻译、总结、问答

2. 对话数据
   格式：conversations
   示例：多轮对话

3. 领域数据
   格式：领域特定格式
   示例：代码、医学、法律
```

### 数据清洗
```
步骤1：去除重复数据
步骤2：过滤低质量数据
步骤3：标准化格式
步骤4：数据增强
步骤5：划分训练/验证/测试集
```

### 数据预处理
```python
def preprocess_function(examples):
    # 拼接instruction和input
    inputs = []
    for instruction, input_text in zip(
        examples["instruction"],
        examples["input"]
    ):
        if input_text:
            inputs.append(f"{instruction}\n{input_text}")
        else:
            inputs.append(instruction)
    
    # Tokenize
    tokenized = tokenizer(
        inputs,
        truncation=True,
        padding="max_length",
        max_length=512
    )
    
    # 设置labels
    tokenized["labels"] = tokenized["input_ids"].copy()
    
    return tokenized
```

### 关键参数说明
```
1. learning_rate：学习率
   - LoRA：1e-4到5e-4
   - QLoRA：1e-4到2e-4
   - 全参数：1e-5到5e-5

2. batch_size：批次大小
   - 根据显存调整
   - 通常2-8
   - 配合gradient_accumulation_steps

3. gradient_accumulation_steps：梯度累积
   - 增加有效批次大小
   - 通常2-16

4. num_train_epochs：训练轮数
   - 通常3-10轮
   - 根据数据量调整

5. warmup_steps：预热步数
   - 学习率预热
   - 通常100-500步
```

## 训练执行

### 基本训练循环
```python
from transformers import Trainer

# 创建Trainer
trainer = Trainer(
    model=model,
    args=training_args,
    train_dataset=train_dataset,
    eval_dataset=eval_dataset,
    tokenizer=tokenizer
)

# 开始训练
trainer.train()

# 评估
eval_results = trainer.evaluate()
print(f"评估结果: {eval_results}")

# 保存模型
trainer.save_model("./final_model")
tokenizer.save_pretrained("./final_model")
```

### 自定义训练循环
```python
import torch
from torch.utils.data import DataLoader
from tqdm import tqdm

# 创建数据加载器
train_dataloader = DataLoader(
    train_dataset,
    batch_size=training_args.per_device_train_batch_size,
    shuffle=True
)

# 训练循环
model.train()
for epoch in range(training_args.num_train_epochs):
    total_loss = 0
    progress_bar = tqdm(train_dataloader, desc=f"Epoch {epoch + 1}")
    
    for batch in progress_bar:
        # 移动数据到设备
        batch = {k: v.to(device) for k, v in batch.items()}
        
        # 前向传播
        outputs = model(**batch)
        loss = outputs.loss
        
        # 反向传播
        loss.backward()
        
        # 梯度裁剪
        torch.nn.utils.clip_grad_norm_(
            model.parameters(),
            training_args.max_grad_norm
        )
        
        # 优化器步骤
        optimizer.step()
        optimizer.zero_grad()
        
        # 更新进度条
        total_loss += loss.item()
        progress_bar.set_postfix({"loss": loss.item()})
    
    # 打印epoch损失
    avg_loss = total_loss / len(train_dataloader)
    print(f"Epoch {epoch + 1} 平均损失: {avg_loss:.4f}")
```

## 评估方法

### 定量评估
```python
# 计算困惑度
eval_results = trainer.evaluate()
perplexity = torch.exp(torch.tensor(eval_results["eval_loss"]))
print(f"困惑度: {perplexity:.2f}")

# 计算准确率（分类任务）
def compute_metrics(eval_pred):
    logits, labels = eval_pred
    predictions = torch.argmax(logits, dim=-1)
    accuracy = (predictions == labels).float().mean()
    return {"accuracy": accuracy.item()}
```

## 最佳实践

### 1. 数据质量
```
原则：高质量数据比大数据量重要
做法：精心设计，人工审核，去除噪声
```

### 2. 超参数调优
```
原则：先小范围测试，再大规模训练
做法：网格搜索，贝叶斯优化
```

### 3. 早停策略
```
原则：防止过拟合
做法：监控验证集性能，性能不再提升时停止
```

### 4. 定期评估
```
原则：持续监控模型性能
做法：定期在验证集上评估，保存最佳模型
```
