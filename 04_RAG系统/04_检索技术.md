# 检索技术

## 检索器类型

### 1. 向量检索（Vector Search）
```
原理：基于嵌入向量的相似度
优点：语义匹配，理解含义
缺点：可能忽略关键词匹配
适用：语义搜索、问答系统
```

### 2. 关键词检索（Keyword Search）
```
原理：基于关键词匹配（TF-IDF、BM25）
优点：精确匹配，速度快
缺点：不理解语义
适用：精确搜索、代码搜索
```

### 3. 混合检索（Hybrid Search）
```
原理：结合向量和关键词检索
优点：兼顾语义和精确匹配
缺点：实现复杂
适用：生产环境、高质量搜索
```

## 混合检索

### BM25 + 向量检索
```python
from langchain.retrievers import EnsembleRetriever, BM25Retriever

# 1. 创建BM25检索器（关键词检索）
bm25_retriever = BM25Retriever.from_documents(chunks)
bm25_retriever.k = 3  # 返回3个结果

# 2. 创建向量检索器
vector_retriever = vectorstore.as_retriever(
    search_kwargs={"k": 3}
)

# 3. 创建混合检索器
ensemble_retriever = EnsembleRetriever(
    retrievers=[bm25_retriever, vector_retriever],
    weights=[0.4, 0.6]  # 40% BM25，60% 向量
)

# 4. 使用
results = ensemble_retriever.get_relevant_documents("机器学习")
```

## 重排序（Reranking）

### 为什么需要重排序？
```
问题：初始检索可能不够精确
解决：对检索结果重新排序

流程：
  1. 初始检索：获取Top-K候选（如Top-20）
  2. 重排序：使用更精确的模型重新排序
  3. 最终结果：返回Top-N（如Top-3）
```

### Cross-Encoder重排序
```python
from sentence_transformers import CrossEncoder
from langchain.retrievers import ContextualCompressionRetriever
from langchain.retrievers.document_compressors import CrossEncoderReranker

# 1. 创建Cross-Encoder模型
cross_encoder = CrossEncoder("cross-encoder/ms-marco-MiniLM-L-6-v2")

# 2. 创建重排序器
compressor = CrossEncoderReranker(model=cross_encoder, top_n=3)

# 3. 创建压缩检索器
compression_retriever = ContextualCompressionRetriever(
    base_compressor=compressor,
    base_retriever=retriever
)

# 4. 使用
results = compression_retriever.get_relevant_documents("机器学习")
```

### LLM重排序
```python
from langchain.retrievers.document_compressors import LLMChainFilter
```

```python
    """使用LLM重写查询"""
    
    prompt = f"""
    将以下查询重写为更清晰、更具体的查询：
    
    原始查询：{query}
    
    重写后的查询：
    """
```

### 3. HyDE（Hypothetical Document Embeddings）
```python
def hyde_search(query, vectorstore, llm):
    """使用HyDE方法搜索"""
    
    # 1. 生成假设性答案
    prompt = f"""
    请为以下问题生成一个假设性的答案：
    
    问题：{query}
    
    答案：
    """
    
    hypothetical_answer = llm.generate(prompt)
    
    # 2. 对假设性答案进行嵌入
    answer_embedding = embeddings.embed_query(hypothetical_answer)
    
    # 3. 搜索相似文档
    results = vectorstore.similarity_search_by_vector(
        answer_embedding,
        k=3
    )
    
    return results
```

## 多模态检索

### 图文检索
```python
from langchain.vectorstores import FAISS
from langchain.embeddings import CLIPEmbeddings

# 1. 创建多模态嵌入
embeddings = CLIPEmbeddings()

# 2. 处理图文文档
documents = []
for image_path, caption in image_caption_pairs:
    # 图片嵌入
    image_embedding = embeddings.embed_image(image_path)
    # 文本嵌入
    text_embedding = embeddings.embed_query(caption)
    
    documents.append({
        "image_path": image_path,
        "caption": caption,
        "embedding": (image_embedding + text_embedding) / 2
    })

# 3. 检索
def search_image(query, top_k=3):
    query_embedding = embeddings.embed_query(query)
    
    # 计算相似度
    similarities = []
    for doc in documents:
        sim = cosine_similarity(query_embedding, doc["embedding"])
        similarities.append((sim, doc))
    
    # 返回Top-K
    similarities.sort(reverse=True)
    return [doc for _, doc in similarities[:top_k]]
```

## 检索评估

### 评估指标
```
1. 召回率（Recall）：相关文档中被检索到的比例
2. 精确率（Precision）：检索结果中相关文档的比例
3. F1分数：精确率和召回率的调和平均
4. MRR（Mean Reciprocal Rank）：第一个相关结果的排名倒数
5. NDCG：考虑排名的相关性分数
```

### 评估代码
```python
def evaluate_retriever(retriever, test_queries):
    """评估检索器性能"""
    
    metrics = {
        "recall": [],
        "precision": [],
        "mrr": []
    }
    
    for query, relevant_docs in test_queries:
        # 检索
        retrieved_docs = retriever.get_relevant_documents(query)
        
        # 计算指标
        recall = len(set(retrieved_docs) & set(relevant_docs)) / len(relevant_docs)
        precision = len(set(retrieved_docs) & set(relevant_docs)) / len(retrieved_docs)
        
        # MRR
        mrr = 0
        for i, doc in enumerate(retrieved_docs):
            if doc in relevant_docs:
                mrr = 1 / (i + 1)
                break
        
        metrics["recall"].append(recall)
        metrics["precision"].append(precision)
        metrics["mrr"].append(mrr)
    
    # 平均指标
    return {
        "avg_recall": np.mean(metrics["recall"]),
        "avg_precision": np.mean(metrics["precision"]),
        "avg_mrr": np.mean(metrics["mrr"])
    }
```
