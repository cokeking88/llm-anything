# 文档处理

## 文档加载

### 支持格式
```
1. PDF文件：PyPDFLoader、PyMuPDFLoader
2. 文本文件：TextLoader
3. 网页：WebBaseLoader
4. Markdown：UnstructuredMarkdownLoader
5. 代码文件：LanguageSpecificTextLoader
```

### 加载示例
```python
from langchain.document_loaders import PyPDFLoader, TextLoader, DirectoryLoader

# 加载单个PDF
loader = PyPDFLoader("document.pdf")
documents = loader.load()

# 加载文本文件
loader = TextLoader("document.txt")
documents = loader.load()

# 批量加载目录
loader = DirectoryLoader("./docs", glob="**/*.txt")
documents = loader.load()
```

### 文档预处理
```python
from langchain.document_loaders import PyPDFLoader
from langchain.text_splitter import RecursiveCharacterTextSplitter

# 1. 加载文档
loader = PyPDFLoader("document.pdf")
documents = loader.load()

# 2. 清理文本
def clean_text(text):
    # 去除多余空白
    text = " ".join(text.split())
    # 去除特殊字符
    text = text.replace("\n", " ").replace("\t", " ")
    return text

# 3. 应用清理
for doc in documents:
    doc.page_content = clean_text(doc.page_content)
```

## 文档分块

### 为什么需要分块？
```
1. Embedding模型有长度限制（通常512-1024 tokens）
2. 小块更精确匹配查询
3. 减少噪声，提高检索质量
4. 避免信息丢失
```

### 分块策略

#### 1. 固定大小分块
```python
from langchain.text_splitter import CharacterTextSplitter

text_splitter = CharacterTextSplitter(
    separator="\n",
    chunk_size=500,
    chunk_overlap=50
)
chunks = text_splitter.split_documents(documents)
```

#### 2. 递归分块
```python
from langchain.text_splitter import RecursiveCharacterTextSplitter

text_splitter = RecursiveCharacterTextSplitter(
    chunk_size=500,
    chunk_overlap=50,
    separators=["\n\n", "\n", " ", ""]
)
chunks = text_splitter.split_documents(documents)
```

#### 3. 语义分块
```python
from langchain.text_splitter import SemanticChunker
from langchain.embeddings import OpenAIEmbeddings

text_splitter = SemanticChunker(
    embeddings=OpenAIEmbeddings(),
    breakpoint_threshold_type="percentile"
)
chunks = text_splitter.split_documents(documents)
```

### 分块参数选择
| 文档类型 | 推荐块大小 | 重叠大小 | 说明 |
|----------|------------|----------|------|
| 一般文档 | 500-1000 | 100-200 | 平衡信息密度和检索精度 |
| 代码 | 300-500 | 50-100 | 保持代码完整性 |
| 书籍 | 1000-2000 | 200-400 | 保持段落完整性 |
| 诗歌 | 100-300 | 20-50 | 保持韵律结构 |
| 技术文档 | 800-1200 | 150-250 | 保持技术完整性 |

### 分块评估
```python
def evaluate_chunks(chunks):
    """评估分块质量"""
    stats = {
        "total_chunks": len(chunks),
        "avg_length": sum(len(c.page_content) for c in chunks) / len(chunks),
        "min_length": min(len(c.page_content) for c in chunks),
        "max_length": max(len(c.page_content) for c in chunks)
    }
    
    # 检查重叠
    overlaps = []
    for i in range(len(chunks) - 1):
        overlap = len(set(chunks[i].page_content) & set(chunks[i+1].page_content))
        overlaps.append(overlap)
    
    stats["avg_overlap"] = sum(overlaps) / len(overlaps) if overlaps else 0
    
    return stats
```

## 文本嵌入

### 什么是嵌入（Embedding）？
```
概念：将文本转换为向量（数字数组）
目的：让计算机理解文本的语义

示例：
  "你好" → [0.2, -0.5, 0.8, ...]（768维）
  "Hello" → [0.3, -0.4, 0.7, ...]（相似向量）
```

### 嵌入模型选择

#### OpenAI嵌入
```python
from langchain.embeddings import OpenAIEmbeddings

embeddings = OpenAIEmbeddings(
    model="text-embedding-ada-002",
    openai_api_key="your-api-key"
)
```

#### 开源嵌入模型
```python
from langchain.embeddings import HuggingFaceEmbeddings

# BGE模型（中文优秀）
embeddings = HuggingFaceEmbeddings(
    model_name="BAAI/bge-small-zh-v1.5"
)

# Sentence-BERT模型
embeddings = HuggingFaceEmbeddings(
    model_name="sentence-transformers/all-MiniLM-L6-v2"
)
```

### 嵌入模型对比
| 模型 | 维度 | 特点 | 适用场景 |
|------|------|------|----------|
| text-embedding-ada-002 | 1536 | OpenAI，效果好 | 生产环境 |
| BGE-large-zh | 1024 | 中文优秀 | 中文任务 |
| all-MiniLM-L6-v2 | 384 | 轻量快速 | 本地部署 |
| E5-large-v2 | 1024 | 多语言 | 多语言任务 |

### 嵌入缓存
```python
from langchain.embeddings import OpenAIEmbeddings

# 设置缓存
```

## 文档增强

### 元数据提取
```python
def add_metadata(documents):
    """为文档添加元数据"""
    for doc in documents:
        # 添加来源信息
        doc.metadata["source"] = doc.metadata.get("source", "unknown")
        
        # 添加长度信息
        doc.metadata["length"] = len(doc.page_content)
        
        # 添加关键词（简单示例）
        keywords = ["LLM", "AI", "机器学习", "深度学习"]
        doc.metadata["keywords"] = [
            kw for kw in keywords if kw in doc.page_content
        ]
    
    return documents
```

### 文档去重
```python
def remove_duplicates(documents):
    """去除重复文档"""
    seen = set()
    unique_docs = []
    
    for doc in documents:
        content_hash = hash(doc.page_content)
        if content_hash not in seen:
            seen.add(content_hash)
            unique_docs.append(doc)
    
    return unique_docs
```

### 文档清洗
```python
def clean_documents(documents):
    """清洗文档"""
    cleaned = []
    
    for doc in documents:
        text = doc.page_content
        
        # 去除多余空白
        text = " ".join(text.split())
        
        # 去除特殊字符（保留基本标点）
        text = text.replace("\x00", "")
        
        # 去除过短的文档
        if len(text) > 50:
            doc.page_content = text
            cleaned.append(doc)
    
    return cleaned
```
