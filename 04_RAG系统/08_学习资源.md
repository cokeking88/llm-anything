## 研究论文

### 基础论文
```
1. Retrieval-Augmented Generation for Knowledge-Intensive NLP Tasks (2020)
   重要性：RAG技术奠基论文
   内容：检索增强生成框架

2. REALM: Retrieval-Augmented Language Model Pre-Training (2020)
   重要性：检索增强预训练
   内容：结合检索的预训练方法

3. Dense Passage Retrieval for Open-Domain Question Answering (2020)
   重要性：密集检索技术
   内容：DPR算法
```

### 进阶论文
```
4. REPLUG: Retrieval-Augmented Black-Box Language Models (2023)
   重要性：即插即用RAG
   内容：无需训练的RAG方法

5. Self-RAG: Learning to Retrieve, Generate, and Critique (2023)
   重要性：自反思RAG
   内容：动态决定何时检索

6. CRAG: Corrective Retrieval Augmented Generation (2024)
   重要性：纠正性RAG
   内容：检索结果验证和纠正
```

第3-4周：技术深入
