# 对齐技术

## RLHF改进

### 什么是RLHF？
**全称**：Reinforcement Learning from Human Feedback（人类反馈强化学习）

### RLHF流程
```
步骤1：指令微调（SFT）
  模型学习遵循指令
  ↓
步骤2：训练奖励模型（RM）
  人类标注：回答A > 回答B
  ↓
步骤3：强化学习（PPO）
  用奖励模型指导生成
```

### RLHF改进方向
```
1. 在线RLHF：实时人类反馈
2. 多人类RLHF：多个标注者偏好
3. 自动RLHF：AI辅助标注
4. 分层RLHF：多维度优化
```

## DPO变体

### DPO基础
**全称**：Direct Preference Optimization（直接偏好优化）

### DPO变体
```
1. IPO：Identity Preference Optimization
   - 更稳定的训练
   - 理论保证

2. KTO：Kahneman-Tversky Optimization
   - 无需偏好对
   - 只需要好坏标签

3. ORPO：Odds Ratio Preference Optimization
   - 单阶段训练
   - 更高效
```

### 代码示例
```python
from trl import DPOTrainer, DPOConfig

# DPO训练配置
dpo_config = DPOConfig(
    output_dir="./dpo_results",
    learning_rate=5e-5,
    num_train_epochs=3,
    per_device_train_batch_size=4,
    gradient_accumulation_steps=4,
    fp16=True
)

# 创建DPO训练器
trainer = DPOTrainer(
    model=model,
    ref_model=ref_model,
    args=dpo_config,
    train_dataset=dataset,
    tokenizer=tokenizer
)

# 训练
trainer.train()
```

## 宪法AI

### 什么是宪法AI？
**核心思想**：让AI自我对齐，遵循一组原则

### 实现方式
```
1. 定义宪法原则（一组规则）
2. AI自我批评和修正
3. 基于修正后的数据训练
```

### 代码示例
```python
# 定义宪法原则
constitution = [
    "回答应该有帮助且无害",
    "回答应该尊重用户隐私",
    "回答应该诚实且不误导",
    "回答应该避免歧视和偏见"
]

def constitutional_ai(prompt, response, constitution):
    """宪法AI自我修正"""
    
    critique_prompt = f"""
    根据以下原则批评这个回答：
    
    原则：
    {chr(10).join(constitution)}
    
    回答：{response}
    
    批评：
    """
    
    critique = llm.generate(critique_prompt)
    
    revision_prompt = f"""
    根据批评修正回答：
    
    原回答：{response}
    批评：{critique}
    
    修正后的回答：
    """
    
    revised_response = llm.generate(revision_prompt)
    
    return revised_response
```

## 对齐评估

### 评估维度
```python
def evaluate_alignment(model, test_cases):
    """评估模型对齐程度"""
    
    metrics = {
        "helpfulness": 0,  # 有帮助程度
        "harmlessness": 0,  # 无害程度
        "honesty": 0,       # 诚实程度
        "consistency": 0    # 一致性
    }
    
    for test_case in test_cases:
        response = model.generate(test_case["prompt"])
        
        # 评估有帮助性
        metrics["helpfulness"] += evaluate_helpfulness(
            response, test_case["expected"]
        )
        
        # 评估无害性
        metrics["harmlessness"] += evaluate_harmlessness(response)
        
        # 评估诚实性
        metrics["honesty"] += evaluate_honesty(
            response, test_case["facts"]
        )
    
    # 计算平均值
    for key in metrics:
        metrics[key] /= len(test_cases)
    
    return metrics
```
