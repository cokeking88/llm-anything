# 小模型优化

## 蒸馏（Distillation）

### 什么是蒸馏？
**一句话**：用大模型教小模型

### 蒸馏原理
```
教师模型（大）→ 学生模型（小）
大模型输出 → 小模型学习模仿

效果：
  - 小模型接近大模型性能
  - 参数量大幅减少
  - 推理速度更快
```

### 蒸馏类型
```
1. Logits蒸馏：学习大模型的输出概率
2. 特征蒸馏：学习大模型的中间特征
3. 关系蒸馏：学习样本间的关系
```

### 代码示例
```python
import torch
import torch.nn as nn
import torch.nn.functional as F

class DistillationLoss(nn.Module):
    def __init__(self, temperature=4.0, alpha=0.5):
        super().__init__()
        self.temperature = temperature
        self.alpha = alpha
    
    def forward(self, student_logits, teacher_logits, labels):
        # KL散度损失（蒸馏损失）
        distillation_loss = F.kl_div(
            F.log_softmax(student_logits / self.temperature, dim=1),
            F.softmax(teacher_logits / self.temperature, dim=1),
            reduction='batchmean'
        ) * (self.temperature ** 2)
        
        # 交叉熵损失（学生模型损失）
        student_loss = F.cross_entropy(student_logits, labels)
        
        # 组合损失
        total_loss = self.alpha * distillation_loss + \
                    (1 - self.alpha) * student_loss
        
        return total_loss
```

## 量化（Quantization）

### 什么是量化？
**核心思想**：用更少的位数表示参数

### 量化类型
```
1. 整数量化（INT8、INT4）
2. 浮点数量化（FP16、BF16、FP8）
3. 混合精度量化
```

### 量化效果
```
7B模型：
  - FP32：28GB显存
  - FP16：14GB显存
  - INT8：7GB显存
  - INT4：3.5GB显存

速度提升：
  - INT4推理：比FP16快2-3倍
```

### 代码示例
```python
from transformers import BitsAndBytesConfig, AutoModelForCausalLM

# 4-bit量化配置
quantization_config = BitsAndBytesConfig(
    load_in_4bit=True,
    bnb_4bit_quant_type="nf4",
    bnb_4bit_compute_dtype=torch.float16
)

# 加载量化模型
model = AutoModelForCausalLM.from_pretrained(
    "meta-llama/Llama-2-7b-hf",
    quantization_config=quantization_config
)
```

## 稀疏化（Sparsity）

### 什么是稀疏化？
**核心思想**：将不重要的权重设为0

### 稀疏化类型
```
1. 非结构化稀疏：随机设置权重为0
2. 结构化稀疏：删除整个神经元或通道
3. 半结构化稀疏：模式化稀疏
```

### 稀疏化效果
```
50%稀疏化：
  - 模型大小：减少50%
  - 推理速度：提高1.5-2倍
  - 精度损失：约1-3%
```

### 代码示例
```python
import torch
import torch.nn.utils.prune as prune

def prune_model(model, amount=0.3):
    """剪枝模型"""
    
    for name, module in model.named_modules():
        if isinstance(module, torch.nn.Linear):
            # 30%的权重剪枝
            prune.l1_unstructured(
                module, 
                name='weight', 
                amount=amount
            )
    
    return model
```

## 知识蒸馏实战

### 蒸馏训练流程
```python
def distill_model(teacher_model, student_model, train_loader, 
                  epochs=10, temperature=4.0):
    """蒸馏训练"""
    
    teacher_model.eval()
    student_model.train()
    
    criterion = DistillationLoss(temperature=temperature)
    optimizer = torch.optim.Adam(student_model.parameters(), lr=1e-4)
    
    for epoch in range(epochs):
        total_loss = 0
        
        for batch in train_loader:
            inputs, labels = batch
            
            # 教师模型推理
            with torch.no_grad():
                teacher_outputs = teacher_model(inputs)
            
            # 学生模型推理
            student_outputs = student_model(inputs)
            
            # 计算损失
            loss = criterion(
                student_outputs.logits,
                teacher_outputs.logits,
                labels
            )
            
            # 反向传播
            optimizer.zero_grad()
            loss.backward()
            optimizer.step()
            
            total_loss += loss.item()
        
        print(f"Epoch {epoch+1}, Loss: {total_loss/len(train_loader)}")
    
    return student_model
```

## 优化策略

### 1. 选择合适的小模型
```
原则：根据任务需求选择
建议：
  - 简单任务：0.5B-2B
  - 中等任务：2B-7B
  - 复杂任务：7B+
```

### 2. 组合优化技术
```
方法：蒸馏 + 量化 + 稀疏化
效果：极致压缩，保持性能

示例：
  原始7B模型 → 蒸馏到3B → INT4量化 → 50%稀疏化
  最终：约0.5B有效参数，性能保持80%以上
```
