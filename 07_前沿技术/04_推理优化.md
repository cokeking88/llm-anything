# 推理优化

## 投机解码（Speculative Decoding）

### 什么是投机解码？
**核心思想**：小模型生成草稿，大模型验证

### 原理
```
小模型：快速生成多个token
大模型：并行验证所有token
效果：加速2-4倍，质量不变

流程：
1. 小模型生成K个token
2. 大模型并行验证这K个token
3. 接受一致的token，拒绝不一致的
4. 从拒绝位置重新开始
```

### 代码示例
```python
class SpeculativeDecoder:
    def __init__(self, draft_model, target_model, k=5):
        self.draft_model = draft_model  # 小模型
        self.target_model = target_model  # 大模型
        self.k = k  # 草稿长度
    
    def generate(self, prompt, max_length=100):
        """投机解码生成"""
        
        tokens = self.tokenize(prompt)
        generated = []
        
        while len(generated) < max_length:
            # 1. 小模型生成草稿
            draft_tokens = self.draft_model.generate(
                tokens + generated,
                max_new_tokens=self.k
            )
            
            # 2. 大模型验证
            verified = self.verify(
                tokens + generated,
                draft_tokens
            )
            
            # 3. 接受验证通过的token
            generated.extend(verified)
            
            # 4. 如果有拒绝，生成新token
            if len(verified) < len(draft_tokens):
                next_token = self.target_model.generate(
                    tokens + generated,
                    max_new_tokens=1
                )
                generated.append(next_token[0])
        
        return self.detokenize(generated)
```

## 分页注意力（Paged Attention）

### 什么是分页注意力？
**核心思想**：像操作系统分页管理内存

### 优势
```
1. 减少内存碎片
2. 支持更大的批处理
3. 提高GPU利用率
4. 支持并发请求

效果：
  - 内存效率：提高2-4倍
  - 吞吐量：提高2-3倍
```

### 实现原理
```
传统：每个序列分配连续内存
分页：将KV缓存分成固定大小的页

优点：
  - 按需分配页
  - 共享未使用的页
  - 避免内存碎片
```

### 代码示意
```python
class PagedAttention:
    def __init__(self, page_size=16):
        self.page_size = page_size
        self.page_table = {}  # 页表
        self.free_pages = set()  # 空闲页
    
    def allocate_pages(self, sequence_id, num_pages):
        """为序列分配页"""
        allocated = []
        
        for _ in range(num_pages):
            if self.free_pages:
                page = self.free_pages.pop()
            else:
                page = len(self.page_table)
            
            allocated.append(page)
        
        self.page_table[sequence_id] = allocated
        return allocated
    
    def attention(self, query, key, value, sequence_id):
        """分页注意力计算"""
        pages = self.page_table[sequence_id]
        
        # 从页中获取KV
        keys = self.load_pages(pages, "key")
        values = self.load_pages(pages, "value")
        
        # 计算注意力
        return compute_attention(query, keys, values)
```

## 连续批处理（Continuous Batching）

### 什么是连续批处理？
**核心思想**：动态添加新请求，不等待整个批次完成

### 优势
```
传统批处理：等待所有序列完成
连续批处理：新请求立即加入

效果：
  - GPU利用率：提高3-5倍
  - 延迟：降低
  - 吞吐量：提高2-4倍
```

### 实现方式
```python
class ContinuousBatching:
    def __init__(self, model, max_batch_size=32):
        self.model = model
        self.max_batch_size = max_batch_size
        self.active_sequences = {}
        self.waiting_queue = []
    
    def add_request(self, request_id, prompt):
        """添加新请求"""
        self.waiting_queue.append({
            "id": request_id,
            "prompt": prompt,
            "tokens": self.tokenize(prompt)
        })
    
    def step(self):
        """执行一步推理"""
        
        # 1. 填充批次
        while len(self.active_sequences) < self.max_batch_size:
            if self.waiting_queue:
                request = self.waiting_queue.pop(0)
                self.active_sequences[request["id"]] = request
            else:
                break
        
        # 2. 并行推理所有序列
        batch_tokens = [seq["tokens"] for seq in self.active_sequences.values()]
        outputs = self.model.generate_batch(batch_tokens)
        
        # 3. 更新序列状态
        completed = []
        for seq_id, output in zip(self.active_sequences.keys(), outputs):
            if self.is_complete(output):
                completed.append(seq_id)
            else:
                self.active_sequences[seq_id]["tokens"].append(output)
        
        # 4. 移除完成的序列
        for seq_id in completed:
            del self.active_sequences[seq_id]
```

## 内存优化

### KV缓存优化
```python
class KVCacheOptimizer:
    def __init__(self, model):
        self.model = model
        self.cache = {}
    
    def optimize_cache(self, sequence_length, batch_size):
        """优化KV缓存"""
        
        # 预计算KV缓存大小
        num_layers = self.model.config.num_hidden_layers
        num_heads = self.model.config.num_attention_heads
        head_dim = self.model.config.hidden_size // num_heads
        
        cache_size = (
            num_layers * 2 *  # K和V
            batch_size *
            sequence_length *
            num_heads *
            head_dim *
            2  # float16
        )
        
        return cache_size
    
    def evict_lru(self, max_cache_size):
        """LRU缓存淘汰"""
        if len(self.cache) > max_cache_size:
            # 移除最久未使用的
            lru_key = min(self.cache.keys(), 
                         key=lambda k: self.cache[k]["last_used"])
            del self.cache[lru_key]
```

## 并行推理

### 张量并行
```python
class TensorParallelism:
    def __init__(self, model, num_gpus=2):
        self.model = model
        self.num_gpus = num_gpus
        self.shard_model()
    
    def shard_model(self):
        """分片模型到多个GPU"""
        # 将模型层分片到不同GPU
        for i, layer in enumerate(self.model.layers):
            gpu_id = i % self.num_gpus
            layer.to(f"cuda:{gpu_id}")
    
    def forward(self, x):
        """并行前向传播"""
        outputs = []
        
        # 每个GPU处理一部分
        for gpu_id in range(self.num_gpus):
            x_gpu = x.to(f"cuda:{gpu_id}")
            output = self.model.layers[gpu_id](x_gpu)
            outputs.append(output)
        
        # 合并结果
        return torch.cat(outputs, dim=-1)
```

## 性能评估

### 评估指标
```python
def evaluate_inference(model, test_prompts):
    """评估推理性能"""
    
    metrics = {
        "latency": [],  # 延迟
        "throughput": [],  # 吞吐量
        "memory_usage": [],  # 内存使用
        "gpu_utilization": []  # GPU利用率
    }
    
    for prompt in test_prompts:
        start_time = time.time()
        
        # 生成
        output = model.generate(prompt)
        
        end_time = time.time()
        
        # 记录指标
        latency = end_time - start_time
        tokens_per_second = len(output) / latency
        
        metrics["latency"].append(latency)
        metrics["throughput"].append(tokens_per_second)
    
    return metrics
```

## 优化建议

### 选择策略
```
场景1：低延迟需求
  - 使用投机解码
  - 使用连续批处理
  - 减小batch size

场景2：高吞吐量需求
  - 使用张量并行
  - 使用连续批处理
  - 增大batch size

场景3：内存有限
  - 使用分页注意力
  - 使用量化
  - 使用KV缓存优化
```
