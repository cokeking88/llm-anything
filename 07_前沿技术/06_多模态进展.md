## 具身智能

### 什么是具身智能？
**定义**：AI在物理世界中行动的能力

### 代表研究
```
1. RT-2：机器人动作 transformer
2. VoxPoser：语言引导操作
3. SayCan：语言到动作
4. MineDojo：开放世界任务
```

### 技术架构
```
感知 → 理解 → 规划 → 行动

感知：多模态传感器输入
理解：场景和任务理解
规划：动作序列规划
行动：物理世界执行
```
